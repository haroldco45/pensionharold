## Uso

Archivo estático único `pension-harold.html`. No requiere servidor ni dependencias locales. Solo abre el archivo en cualquier navegador moderno.

```
git clone https://github.com/haroldco45/pension-harold
cd pension-harold
# Abrir pension-harold.html en el navegador
```

También puede publicarse en **GitHub Pages** activando la rama `main` como fuente en Configuración → Pages.
